# Raylib test

A small learning experiece using C and [raylib](https://github.com/raysan5/raylib) to build GUI applications.

## Setup

### Prerequisites

- Meson
- Ninja
- Modern C compiler


## Build

1. Setup build directory

```shell
meson setup -C builddir
```

2. Compile programm

```shell
meson compile -C buildddir
```
